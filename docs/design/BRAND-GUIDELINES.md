# IntelligentSPM Brand Guidelines

## Logo Treatments

### Primary Logo
- **IntelligentSPM**: White "Intelligent" + Teal "SPM"
- Colors: `#E2E8F0` + `#38BDF8`

### Personal Brand
- **The Toddfather**: White "The" + Orange "Toddfather"
- Colors: `#E2E8F0` + `#FF8737`

### Dynamic Logo (per topic)
The "SPM" color changes based on content area:
| Topic | SPM Color |
|-------|-----------|
| Clearing House / General | Teal `#38BDF8` |
| AI / Comp Plan | Purple `#8241C8` |
| Governance | Lime `#A3E635` |
| AskSM | Orange `#FF8737` |

---

## Color Palette

### Primary Colors
| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Background Dark | `#0F172A` | 15, 23, 42 | Page backgrounds |
| Background Light | `#1E293B` | 30, 41, 59 | Cards, sections |
| Teal | `#38BDF8` | 56, 189, 248 | Primary brand, "SPM", CTAs |
| Orange | `#FF8737` | 255, 135, 55 | "Toddfather", pricing, accents |
| AI Purple (Primary) | `#8241C8` | 130, 65, 200 | AI badges, icons |
| AI Purple (Dark) | `#58108E` | 88, 16, 142 | AI backgrounds, shadows |

### Semantic Colors
| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Lime | `#A3E635` | 163, 230, 53 | Success, positive, governance |
| Hot Pink | `#EA1B85` | 234, 27, 133 | Errors, warnings, problems |
| Text Primary | `#E2E8F0` | 226, 232, 240 | Headlines, body text |
| Text Muted | `#94A3B8` | 148, 163, 184 | Secondary text, captions |

### Rule: Purple = AI
Always use purple (`#8241C8` or `#58108E`) for AI-related features, badges, and content.

## Button Styles

### Primary CTA
- Background matches hero accent color
- White text (`#FFFFFF`)
- Padding: 18px 36px
- Border radius: 10px
- Font weight: 700

### Secondary CTA (Snarky "Book a...")
- Transparent background
- White border and text
- All use "Book a [Snarky Term]" format
